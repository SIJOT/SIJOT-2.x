Permissions: 
===============

We have build some permission role's into the project. You can find info about each role below:

- [Permissions overview]()

## Permissions overview.

#### leden-beheer

Has permission for: 

- Block and unblock a user. 
- Delete a user. 
- Listing view for all the user. 
- Register a new login.
